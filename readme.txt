COMMANDS:
enc
  calls encfile
  encrypts the file at path with key
  fmt: enc path key
  @staticmethod
dec
  calls decfile
  decrypts the file at path with key
  fmt: dec path key
nf
  calls createfile
  creates new file at path
  fmt: nf path
rf
  calls readfile
  reads file at path
  fmt: rf path
wf
  calls writefile
  writes to file at path. enter text to write on nextline.
  fmt: wf path
