COMMANDS:
enc
  encrypts the file at path with key.
  fmt: enc path key
dec
  decrypts the file at path with key.
  fmt: dec path key
nf
  creates new file at path.
  fmt: nf path
rf
  reads file at path.
  fmt: rf path
wf
  writes to file at path. enter text to write on nextline.
  fmt: wf path
df
  deletes file at path. it will only delete a file outside of the xorencryptor if you give the absolute path.
  fmt: df path
exit
  exits program
  fmt: exit
